# Monstera-UI

React UI Library

-   git clone [project repo]
-   npm install (possibly: npm install --force)
-   npx storybook@latest init

In some cases:

-   yarn install --ignore-engines

Run project:

-   npm run storybook

Build project

-   Update version in package.json
-   npm run build

Update npm library

-   npm publish --access public
